
# Graphic Design

This documents contains general rules regarding the graphic design of the project.

## Feel

This project is heavily inspired by high-quality scrollytelling projects such as the ones produced by The Pudding.

The visual aspects should aim towards simplicity and aesthetically pleasing minimalism.

## Cards

Visualizations and images should appear within cards with color Pantone Seedpearl or # E6DAC4.

Cards should have rounded corners.

Standalone text steps should have bold typefaces with added shadow for contrast and effect.

## Colors

The color palette is inspired by classic european style boardgames. The accent colors are muted versions of wooden boardgame pawns: green, blue, red, yellow, purple.
- Green: # 4a7a61
- Blue: # 3b6b88
- Red: # b84a39
- Yellow: # d19e2b
- Purple: # 7a4b75

The cards that contain the images and visualizations should be Pantone Seedpearl or # E6DAC4.

## Visualizations

Visualization backgrounds should be # fef8ef.

Visualizations should be minimal in design, aiming at beauty and elegance. A general reference for visualizations can be found at /reference_images/viz_visual_reference:

![[viz_visual_reference.png]]


## Images

Images must not be distorted or stretched in any way.

Whenever images have size ratios over 1.75:1, the longer side (height or width) should be cropped so that the the ratio stays at 1.75:1, keeping the center at the center.

Apply a mask so that every image has rounded corners.
